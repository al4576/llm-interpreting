2.2

Embedding Lookup

Token IDs index rows of embedding matrix E. Each row is a vector in R^d.

E ∈ R^(V x d) e_i = E[t_i] h_i^(0) = e_i + p_i

in: [n] IDs   out: [n, d] vectors

2.3

Self-Attention

Each token produces Query, Key, Value. Attention is a weighted sum of values, weighted by key/query similarity.

Q = HW_Q, K = HW_K, V = HW_V S = (QK^T) / sqrt(d_k) A = softmax(S) Z = AV

in: [n, d]   out: [n, d] (context-mixed)

Figure 2.3   Rows = query tokens, columns = keys. Outlined row shows one token attending mostly to keys 3 and 4.

2.4

Next-Token Sampling

Final hidden state is projected to logits over the vocabulary; softmax gives probabilities; decoding chooses.

z = h_last W_out p_i = exp(z_i / T) / Σ_j exp(z_j / T) t_next ~ Categorical(p)

in: [d] final vector   out: 1 sampled token
